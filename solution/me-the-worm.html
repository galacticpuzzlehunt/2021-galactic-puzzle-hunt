Solution Me, the Worm
Answer EAT THIS BOOK

Written by Jenna Himawan, Chris Jones, and Azalea Weisblat

This puzzle is an absurd choose-your-own-adventure novel about being an earthworm. You can read the full text here. Reading through the book1, we can see that there are several possible endings, and also on some pages there are objects with an asterisk (*) on them. Author’s Notes

This puzzle went through a lot of phases before it arrived in its final state.

It was the first puzzle assigned an answer in the concert round. Back then, it was a text adventure with some persistent state. After four minutes, you would have to restart the text adventure, but some things (switches being flipped, doors being open, etc) might remain the same. We ideated a lot of puzzle mechanics for this idea, but we never really got to a point of a puzzle we were happy with.

So we went on to a second idea. We were assigned the answer EAT THIS BOOK, and we decided that naturally the best thing to do would be to write a puzzle that was a book whose reader was a bookworm. In this concept, the puzzle was a choose-your-own-adventure book like the final puzzle, but there were several subpuzzles that culminated in actions that the player character (also the author of the book) could take to change the book, like cutting out pages, burning pages (Fire Ant accidentally setting the book on fire), cutting holes in pages (Bookworm chewing holes), etc. We had a good chunk of this book brainstormed and had started to write it before we lost motivation and were distracted by finishing up the rest of the puzzles in the hunt. A week before hunt, this puzzle still hadn‚Äôt been written and we were lacking the motivation to implement and write something this complicated... but we needed a puzzle.

So Chris brainstormed changes and came up with version three, the extraction you see today. Jenna joined in at this point to help postprod the puzzle and write story text.

The puzzle was testsolved for the first time on the night of Tuesday, August 23 (3 days before hunt). The testsolve was originally scheduled for 10pm ET, then got pushed back because the puzzle was not done. Eventually it started at 1am ET. This first testsolving group got stuck because we weren‚Äôt cluing the extraction hard enough. This was pretty stressful, since we only had 2 more days to make the puzzle work.

On Wednesday, a new extraction scheme was proposed (using only the layers of the Earth instead of using items from each layer). However, we weren‚Äôt convinced that we had time to implement the change, nor that it was actually an improvement, so we settled for cluing the existing extraction harder.

The puzzle probably could have been more elegant with more testing and more iterations. But many people enjoyed the puzzle (we hope you will too!), and that‚Äôs the part that matters.

Jenna: Writing this puzzle was a roller-coaster. It was so last minute that I sometimes got anxious or frustrated. At the same time, I loved the puzzle and was really happy to see testsolvers playing it. tl;dr - extremely fun AND extremely stressful

Azalea: I got pretty demotivated on puzzle things and busy with real life around the time when we were trying to write the second version of the puzzle. I appreciated Chris taking that in stride and ideating a replacement idea on his own, and I appreciated Chris and Jenna doing the brunt of the writing while I wasn‚Äôt feeling up to it. I‚Äôm really happy with the puzzle we ended up with.

Chris: It was surprising that thinking of the extraction actually took significantly longer than writing the book. The book was furiously written in two-ish days (not recommended!). Praise the Holy Worm!
