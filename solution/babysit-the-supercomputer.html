Solution Babysit the Supercomputer
Answer BYE BYE BABYLON

Written by Andy Hauge and Jenna Himawan

This puzzle is presented as an interactive game. We are told that we need to finish “The Final Computation” and are presented with some buttons to schedule various tasks.

However, if we try to schedule The Final Computation, it takes 6 minutes! Unfortunately, the asteroid will strike after 4 minutes, and The Final Computation will not have time to finish.

If we try playing around with scheduling the tasks, we might notice that, when a task completes, it may decrease the runtimes of other tasks. The interface shows a glow around the runtimes of the tasks which were affected by the most recent task completion.

There are various ways to schedule tasks such that The Final Computation is completed. One such scheduling is written below. (You can also see the Appendix for a screencast.)

  1. BETA and GAMMA simultaneously

  2. EPSILON

  3. THETA

  4. IOTA

  5. DELTA

  6. ALPHA

  7. ZETA and KAPPA simultaneously (both begin after ALPHA completes)

  8. LAMBDA

  9. ETA

  10. The Final Computation

If we successfully complete The Final Computation, we will get a message saying that the “expected output” was ALFALFA AMALGAMS.

For the next part of the puzzle, we’ll need to pay attention to the “supercomputer output”. As the computer runs, it occasionally outputs strings. The first output is always the input word. It turns out that a computation will output something iff its capacity is 3.

There is a unique ordering of these 3-capacity tasks:

  1. DELTA

  2. LAMBDA

  3. ETA

  4. The Final Computation

Each one of these tasks applies a particular transformation to the last output. The transformations are as follows:

  1. DELTA swaps adjacent letters of the alphabet, so A ‚Üî B, C ‚Üî D. Example: BOAT ‚Üí APBS.

  2. LAMBDA shifts backward by 14 any letter whose position in the alphabet is ‚â• 15 (i.e. O and all the letters after). Example: BOAT ‚Üí BAAF.

  3. ETA replaces B with AM. Example: BOAT ‚Üí AMOAT.

  4. The Final Computation shifts the 4th-to-last letter by -4 and the last letter +6. Example: BOAT ‚Üí XOAZ.
