Solution World's End Concert
Answer AIR HORNS

Written by Chris Jones and Seth Mulhall

The primary a-ha of this metapuzzle is that it refers to the biblical apocalypse, and specifically to the events described in the Book of Revelation. Each of the puzzle answers refers to a specific event in Revelation. On the metapuzzle itself, this is clued by “divine Madonna” (Madonna is another name for Virgin Mary) and the emphasis on the end of the world. Confirmation for the Book of Revelation is also provided by the word “reveal” in the flavor text.

Each answer refers to a specific verse of Revelation. The music video has 8 blank “verses” which we can match up with one of the verses, using the King James Version of the Bible. Some of the events are associated with multiple verses; we should pick the one with the strongest association (the underlines confirm which verse to use and give an ordering to the answers). This gives us the concept, chapter, and verse columns in the table below.

Finally, we need to use the notes in the music video. The number of notes on screen at any time is between 1 and 8, which is a reasonable range for indices into the puzzle answers, and it changes every 5 seconds. How can we associate a time to each answer/verse?

Biblical verses are typically written using a colon, e.g. 1:8 and 16:16. This is the same format as a timestamp, and this suggests that for each verse, we take the index at the corresponding time (chapter/minutes should be taken modulo 4, since the concert happens “again and again”).

One difference is that the leading zero is omitted in the seconds place when referring to a Bible verse, e.g. 1:8 instead of 1:08. The timer on the metapuzzle page is modified so that it doesn’t show the leading zero either.

Indexing into the puzzle answers in the order of the music video yields the answer AIR HORNS, which are something you might need to play a concert in space.
