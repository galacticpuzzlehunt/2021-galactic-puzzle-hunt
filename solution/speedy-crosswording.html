Solution Speedy Crosswording
Answer BENJAMINS

Written by Jack Chen, Jenna Himawan, and Jakob Weisblat

Initial observations

When we open this puzzle, we see an interactive crossword player. Given the title of the puzzle, we might wonder what happens if we can successfully fill the grid.

It turns out that if you fill the first grid, a second crossword is shown!

The puzzles are always the same, and always shown in the same order, across time loops. To successfully fill all the grids, we can iteratively type in the answers to the crosswords we’ve already solved, then copy the new crossword to work on it outside of the webpage.

As we try to solve these crosswords, we may notice that some entries seem a little off. In fact, each crossword has a particular gimmick.

The solutions to the crosswords and their gimmicks are as follows:

Extracting from each crossword

Once we have solved all the crosswords, we are shown an image and some numbers:

The image provides some clues about what to do with the gimmicked cells in each crossword.

Puzzle #1: Some cells contain multiple letters

The gimmicked cells in this crossword are: RK AR KR RO SP PA OC CK. (See the yellow squares in the solution grid above.) As hinted by the overlapping rectangles in the extraction photo, we can “link” these bigrams so that the end of one will be the start of the next: SP PA AR RK KR RO OC CK After eliminating duplicates of adjacent letters, we get a cluephrase: SPARK ROCK.

Putting it all together

Now, we need to use the numbers that we were given: 12223111 332 441344. We can answer the cluephrase from crossword i with an n-letter word in such a way that there are n occurrences of i in the string of numbers. For example, there are five 1s in 12223111 332 441344, so we can answer Puzzle #1’s cluephrase with a five-letter word.

Author’s Notes

One month before the start of GPH, Jenna killed this puzzle. Only part of a (probably mediocre at best) crossword had been written, none of the tech implementation was done, a lot of other things needed to happen for hunt, and she thought the puzzle could not be finished.

Jakob (who at that point was an editor for the puzzle) volunteered to help by constructing crosswords. This allowed the puzzle to brought back from the dead.

The crosswords themselves were written over the course of a day during a family vacation; while Jakob wrote most of the clues, they’d like to thank their parents and brothers for contributing a few here and there and for putting up for repeated requests like “what’s an interesting and difficult clue for ENDS?” on an hours-long drive.

Jenna implemented the web part of the puzzle, adapting much of the code from Fencing from Mystery Hunt 2021, which is publicly available on GitHub. We are profoundly grateful that so much work had already been put into it and that the code was available for our reference.
